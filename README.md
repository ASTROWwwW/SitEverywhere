# **Sitting System – Smart Chairs & Wall Leaning (FiveM)**

Système d’assise optimisé pour FiveM permettant aux joueurs de s’asseoir sur **toutes les chaises, bancs, canapés** du jeu ainsi que de se **poser contre les murs** avec détection automatique.
Compatible **ox_target**, entièrement configurable, fluide et sans bugs.

### **Fonctionnalités**

* Détection automatique de **toutes les chaises supportées** (plus de 200 modèles configurés).
* Gestion des **multi-sièges** : bancs et canapés avec plusieurs positions.
* Réservation serveurs des places pour éviter les conflits.
* Assise fluide via **scénarios réalistes**.
* Fonction **wall-lean** (se poser contre un mur) avec raycast intelligent.
* Rotation instantanée et repositionnement propre sur le siège.
* Système **anti-glitch** : ragdoll, mort, déplacement forcé = dégagement propre.
* Contrôles désactivés pendant l’assise pour éviter les exploits.
* Entièrement configurable via `cfg_sit.lua`.

### **Points forts**

* **Ultra optimisé** (0.00 ms idle).
* **Standalone**, compatible ESX / QBCore / Standalone.
* Parfait pour bar, restaurants, appartements, salons RP.
* Support complet des objets custom (option configurable).
